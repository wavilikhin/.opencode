description: Commit changes and push to remote
---

Follows the same process as `/commit`, then pushes the changes to the current branch on the remote repository.

## Instructions

### Execute commit process:
Follow all instructions from `/commit` command to create the commit.

### After successful commit:
Push the changes to the current branch:

```bash
git push
```

If the remote branch doesn't exist or there are any issues, inform the user.
